# README

## Project setup

```
npm install
```

## Run locally

```
npm run dev
```

## Check before PR

```
npm run pr
```

## Testing & coverage

### Run unit tests once

```
npm run test:unit
```

### Generate coverage report

```
npm run test:coverage
```

### Run unit tests + coverage Vitest GUI mode

```
npm run test:dev
```

## Format

```
npm run format
```

## CI (lint + tests)

```
npm run ci
```

## Build + CI

```
npm run build:full
```
